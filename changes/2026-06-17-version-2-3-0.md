# Truthmark 2.3.0

Previous version: 2.2.1
New version: 2.3.0

## Rationale

Release the canonical agent workflow package and generated-surface adapter migration as the next minor version.

## Release payload

- Add `.truthmark/agent/` as the canonical generated workflow package with manifest metadata and deterministic hashes.
- Convert adapter-mode hosts to reference canonical workflow files instead of duplicating workflow body prose.
- Add generated-surface freshness diagnostics for canonical package files, adapter references, expanded-adapter hashes, and duplicate adapter prose.
- Refresh package metadata and generated runtime markers for Truthmark 2.3.0.
