# Truthmark 2.3.0

Previous version: 2.2.1
New version: 2.3.0

## Rationale

Release the canonical agent workflow package and host-native generated-surface package migration as the next minor version.

## Release payload

- Add `.truthmark/agent/` as the canonical generated workflow package with manifest metadata and deterministic hashes.
- Render host skill directories as native generated packages with colocated procedure, report-template, helper, and lease resources.
- Add generated-surface freshness diagnostics for canonical package files and host-native package entry/support files.
- Refresh package metadata and generated runtime markers for Truthmark 2.3.0.
