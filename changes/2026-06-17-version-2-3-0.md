# Truthmark 2.3.0

Previous version: 2.2.1
New version: 2.3.0

## Rationale

Release the host-native generated-surface package migration as the next minor version.

## Release payload

- Generate host-native skill packages with colocated support files and helper metadata so supported agent hosts can package skills without depending on a separate repo-local workflow copy.
- Render host skill directories as native generated packages with colocated procedure, report-template, helper, and lease resources.
- Add generated-surface freshness diagnostics for host-native package entry/support files.
- Refresh package metadata and generated runtime markers for Truthmark 2.3.0.
