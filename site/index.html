Install in minutes.

Truthmark is local-first and Git-native. It writes repo-local routing, truth docs, and AI-host instructions that travel with your branch.

$ cd /path/to/your-repo
$ npm install -g truthmark
$ truthmark config
$ truthmark init
$ truthmark check
Truthmark is intentionally small: local, committed, branch-scoped, and reviewable.
